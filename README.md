# Orientamento ISIAU

Repository ufficiale del sito di orientamento di ISIA Urbino.

Il repository è stato creato per consolidare e migliorare la gestione del sito di orientamento di ISIA Urbino.

Il sito è costruito con **HTML, Tailwind CSS e DaisyUI**, garantendo un design moderno, responsivo e performante. Questa scelta tecnologica permette massima flessibilità nella presentazione dei contenuti e nell'esperienza utente, mantenendo al contempo pieno controllo sui dati e sull'infrastruttura.

Il sito è pensato per accompagnare studenti e famiglie nella scoperta dell'offerta formativa di ISIA Urbino, fornendo informazioni chiare e accessibili sui corsi, le procedure di ammissione e la vita accademica.

**Sito pubblicato:** [orientamento.isiaurbino.net](https://orientamento.isiaurbino.net]
